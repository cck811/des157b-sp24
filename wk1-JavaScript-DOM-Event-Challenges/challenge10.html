Challenge 10

Scroll and Watch the Picture Change

Challenge Directions

Hello! The key to understanding this challenge is by looking at the source attribute for the image from the placekitten API. The two numbers at the end deterimine the size of the picture, but they also change the picture. If you change the src to http://placekitten.com/301/301, you will see you get a different image.

The challenge is to make it so that as the page scrolls down, the numbers in that source attribute change and the picture changes. I have set the page to 10,000 pixels tall, and I have fixed the position of the content at the top of the window. Also the image is set to display at 300x300 pixels no matter what the source attribute says.

  1. To make this work, you will need three variables, plus one to which you can assign the image element. The three variables are pageTop, prevPageTop, and size. Use the let keyword and assign 0 to all of them. Then add the variable for the img element.
  2. Then you will need to attach an event listener to the window and listen for the scroll event. When that fires, run an anonymous function.
  3. In the anonymous function set pageTop to window.pageYOffset if you console.log() out pageTopyou will see numbers going up and down in the console as you scroll the page.
  4. It is probably not a good idea to change the picture every time you scroll a tiny little bit. So instead, check if pageTop is divisible by 30 using the modulus operator. If it is, increment the size variable.
  5. Then you can change the source attribute of the image so that the numbers at the end are 300 + size.
  6. What you have should be working, but it would be nice if the size variable incremented as you scrolled down the page (and pageTop gets larger) and decrements as you scroll up the page. This is what the prevPageTop variable is for. You can use it to compare it's value to pageTop. If it's a smaller number than pageTop, then you are scrolling down, and size should be incremented. If you are scrolling up, then prevPageTop will be larger, so size shoudl be decremented. See if you can figure it out. Scroll events are always challenging.
