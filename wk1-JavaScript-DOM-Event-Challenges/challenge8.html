Challenge Directions

  1. Create a variable and assign it all the input elements in the document using an element method.
  2. The variable you created will be an array of input elements. You need to loop through that array and add an event listener to each checkbox in the array.
  3. Within the loop, add an event listener for each checkbox that captures a click event and then runs an anonymous function, passing the special event property into this function.
  4. Use the event property to target the checkbox that was clicked and use the parentElement property to get to the parent list item and assign that to a variable, which could be called thisListItem.
  5. Once you have the parent list item, you can set it's textDecoration to line-through and color to light grey, as you did in challenge 7.
  6. Bonus Challenge: Can you make it so that when you click the checked boxes they go back to textDecoration="none" and color="#333"?

Hints: There are three different loops you could use. You could use a for loop. You could use a for.. of loop, or my prefered option is to use the forEach() array method.

The event object can be passed into the anonymous function that is called when the event happens. You can learn about that more on this paage.
