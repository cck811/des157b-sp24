Challenge 6

Replace the text in this paragraph with the best thing that has happened for you this week.

Challenge Directions

  1. Create a variable, declared with const, that holds a string of text indicating the best thing that has happened for you this week.
  2. Create a variable, declared with const, that is assigned the paragraph with the ID set to "target" from the page using an element method.
  3. Use an element property or node property to replace the text in the paragraph with the text assigned to the variable you created in the first step.
  4. Use an element style property to set the color of the text in the paragraph to green.

Hint: Look at this page for a refresher on element properties. If you want to try to use a node property instead, you might check this page. To find out more about the element style property, you can check this page.
