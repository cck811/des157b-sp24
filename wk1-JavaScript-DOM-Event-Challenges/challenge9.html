Challenge 9

Your Optimal Band Name

hello!

Challenge Directions

  1. Make a variable and use an element method to assign the form to it.
  2. Attach an event listener to the variable you just created and capture the submit event and add an anonymous function. Be sure to pass in the event object and use preventDefault() to keep the form from submitting when the submit button is pressed
  3. Create 4 variables to get the data from each of the fields in the form and assign each piece of data its corresponding variable.
  4. At the bottom of the anonymous function that runs when the submit event happens, call two functions that you will write next. The first is called updatePage() and it takes 4 arguments: one for each field. The second is called emptyFields() and it will just clear out the form.
  5. The updatePage() function takes 4 arguments: name, verb, food and type. Construct a string that consists of a heading that says, "Your Optimal Band Name", followed by a paragraph that includes the bandname constructed like this: name and the verb food type. At the end of this function, use an element property to insert this string into the div on the page.
  6. The second function to write is called emptyFields() and it should use a loop to empty out the values in the text fields, so that the form is ready to use again. (Don't worry about resetting the select field)

Hints: There are a few ways to get data from the fields. You can get data from each field individulally by acessing the value property of that field in combination with an ID for the input tag. You can use a loop, and loop through all the input tags and get the values out (intermediate skill level). Or you can use the formData() API and the get() method. You can find out more about it on this page.
