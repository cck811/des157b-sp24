Challenge 5

Inspect the page and view the console to see the results of this challenge.

Challenge Directions

  1. As with example 4, start with a variable that holds a string containing a famous quote.
  2. Then create a function expression assigned to a variable called words that takes the string as an argument and returns an array where each element in the array holds each word from the string.
  3. Then create a named function called processWords() that takes an array as an argument. This function will output the the console an incrementing list of numbers with each word from the array. After that it will output the console a statement indicating how many words are in the array.
  4. Call the processWords() function passing in the words you created.

Hint: Look for a string method that will divide words up based on a character you pass it, like a space character, and put them in an array. Then also review how function expressions work in JavaScript.
