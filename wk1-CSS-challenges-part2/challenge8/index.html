Fix The Boxes

For this challenge, fix the element with the class "boxes" so that there are four boxes in a row. Each box should be square, and remain square even if the browser is narrower and there is less horizontal space. There should be a 20px space between each box. The numbers in the boxes should be in the vertical and horizontal center.
